# Finding your preferred city


## Use Case 1:
Based on inputs from the user (preferred temperature, rain, pollen allergy, ...), determines the best city for you to be over the next 10 days.

## Use Case 2:
Create / Update dashboard of cities clustered by climate profiles.

## How to 
Run "python3 main.py forecast" for use case 1, or "python3 main.py archive" for use case 2.
